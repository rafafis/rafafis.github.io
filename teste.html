Vamos fazer uma luz piscar com o Arduino!

O Arduino é como um pequeno cérebro eletrônico. Com ele, podemos controlar luzes, sons e muito mais!

Hoje, vamos aprender a fazer uma luz piscar: acender por um segundo e apagar por um segundo, repetidamente.

O que você precisa:

  • Um Arduino (como o Arduino Uno)
  • Um cabo USB para conectar o Arduino ao computador
  • Um LED (luzinha) e um resistor (se quiser usar um LED externo)

O código para fazer a luz piscar:


const int LED_PIN = 13; // O pino 13 já tem uma luz no Arduino

void setup() {
  pinMode(LED_PIN, OUTPUT); // Dizemos que o pino 13 será usado para saída
}

void loop() {
  digitalWrite(LED_PIN, HIGH); // Acende a luz
  delay(1000);                 // Espera 1 segundo (1000 milissegundos)
  digitalWrite(LED_PIN, LOW);  // Apaga a luz
  delay(1000);                 // Espera mais 1 segundo
}

Entendendo o código:

setup(): Esta parte acontece uma vez quando o Arduino é ligado. Aqui, dizemos que vamos usar o pino 13 para controlar uma luz.

loop(): Esta parte acontece várias vezes, sem parar. Primeiro, acendemos a luz, esperamos 1 segundo, apagamos a luz e esperamos mais 1 segundo. Depois, tudo se repete!

Experimente!

Conecte seu Arduino ao computador, abra o programa do Arduino (IDE), cole o código acima e envie para o Arduino. Você verá a luz piscando!
